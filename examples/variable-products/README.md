# Virtual Display – Variable Products

This demo shows how to integrate a 3D model with variable product attributes that influence the model's appearance and behavior.

- **Attribute-driven configuration:** Change colors, materials, and components in real-time
- **Perfect for configurable products** like furniture, vehicles, or customizable items
- **Live state synchronization** between UI controls and 3D model
- **Full source code included:** See exactly how to implement product variations

## Try it

Run it using pnpm:
```bash
pnpm dev
```

Open `http://localhost:4002/` in your browser.

## What does it show?

- How to connect product attributes to 3D model variations
- Real-time synchronization between form controls and model state
- Implementation of configurable product options (materials, colors, components)
- State management for complex product configurations

See `/src/index.ts` for the integration code.
